Import Alibaba Cloud AccessKeyID/PW to your local PC 

1) Download the Alibaba Cloud CLI & Set Environment Path
Ref to : https://www.alibabacloud.com/help/en/alibaba-cloud-cli/latest/windows
2) Configure AK Profile
 - aliyun configure --mode AK --profile <AnyRandomProfileName>
 - aliyun configure --profile <AnyRandomProfileName>
Ref to : https://www.alibabacloud.com/help/en/alibaba-cloud-cli/latest/interactive-configuration#section-5pj-p7j-06z
